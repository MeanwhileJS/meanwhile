## [Border Color](https://tailwindcss.com/docs/border-color)

```js
borderColor: ({ theme }) => ({
    ...theme('colors'),
    DEFAULT: theme('colors.gray.200', 'currentColor'),
}),
```
Sets the color of the border. It can be customized using theme colors with a default value. [MDN Border Color](https://developer.mozilla.org/en-US/docs/Web/CSS/border-color)

## [Border Opacity](https://tailwindcss.com/docs/border-opacity)

```js
borderOpacity: ({ theme }) => theme('opacity'),
```
Sets the opacity of the border color. [MDN Border Opacity](https://developer.mozilla.org/en-US/docs/Web/CSS/opacity)

## [Border Radius](https://tailwindcss.com/docs/border-radius)

```js
borderRadius: {
    none: '0px',
    sm: '0.125rem', // 2px
    DEFAULT: '0.25rem', // 4px
    md: '0.375rem', // 6px
    lg: '0.5rem', // 8px
    xl: '0.75rem', // 12px
    '2xl': '1rem', // 16px
    '3xl': '1.5rem', // 24px
    full: '9999px',
},
```
Sets the radius of the element's corners. [MDN Border Radius](https://developer.mozilla.org/en-US/docs/Web/CSS/border-radius)

## [Border Spacing](https://tailwindcss.com/docs/border-spacing)

```js
borderSpacing: ({ theme }) => ({
    ...theme('spacing'),
}),
```
Sets the spacing between borders of adjacent elements. [MDN Border Spacing](https://developer.mozilla.org/en-US/docs/Web/CSS/box-shadow)

## [Border Width](https://tailwindcss.com/docs/border-width)

```js

    DEFAULT: '1px',
    0: '0px',
    2: '2px',
    4: '4px',
    8: '8px',
},
```
Sets the width of the border. [MDN Border Width](https://developer.mozilla.org/en-US/docs/Web/CSS/border-width)
