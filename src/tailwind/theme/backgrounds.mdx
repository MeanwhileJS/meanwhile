## [Background Color](https://tailwindcss.com/docs/background-color)

```js
backgroundColor: ({ theme }) => theme('colors'),
```
Sets the background color of an element using theme colors. [MDN Background Color](https://developer.mozilla.org/en-US/docs/Web/CSS/background-color)

## [Background Image](https://tailwindcss.com/docs/background-image)

```js
backgroundImage: {
  none: 'none',

  'gradient-to-r': 'linear-gradient(to right, var(--tw-gradient-stops))',

```
Sets the background image of an element, including gradients. [MDN Background Image](https://developer.mozilla.org/en-US/docs/Web/CSS/background-image)

## [Background Opacity](https://tailwindcss.com/docs/background-opacity)

```js
backgroundOpacity: ({ theme }) => theme('opacity'),
```
Sets the opacity level for the background color. [MDN Background Opacity](https://developer.mozilla.org/en-US/docs/Web/CSS/opacity)
