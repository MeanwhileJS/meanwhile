## [Animation](https://tailwindcss.com/docs/animation)

```js
animation: {
    none: 'none',
    spin: 'spin 1s linear infinite',
    ping: 'ping 1s cubic-bezier(0, 0, 0.2, 1) infinite',
    pulse: 'pulse 2s cubic-bezier(0.4, 0, 0.6, 1) infinite',
    bounce: 'bounce 1s infinite',
},
```
Defines the animations with their names and keyframe properties. [MDN Animation](https://developer.mozilla.org/en-US/docs/Web/CSS/animation)

## [Keyframes](https://tailwindcss.com/docs/keyframes)

    },
  },
  pulse: {
    '50%': {

    },
  },
  bounce: {
    '0%, 100%': {

      animationTimingFunction: 'cubic-bezier(0.8,0,1,1)',

    '50%': {
      transform: 'none',
      animationTimingFunction: 'cubic-bezier(0,0,0.2,1)',

Defines the keyframes for the animations. [MDN Keyframes](https://developer.mozilla.org/en-US/docs/Web/CSS/@keyframes)
